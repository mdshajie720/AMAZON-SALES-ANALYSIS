## Getting Started

First, run the development server:

```bash
npm install
npm install fast-glob
npm install fastq
 npm install fast-equals

npm run dev
# or
yarn dev

# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
